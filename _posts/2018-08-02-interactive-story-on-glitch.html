I’ve been experimenting using Glitch.com recently - it’s an amazingly powerful tool hidden in a humorous skin. Like Heroku and similar tools, sites/apps get spun up without you needing to know anything about the infrastructure stuff that’s going on in the background - the bit that makes Glitch special is that you can remix other people’s code really easily, and again, spin it up quickly with it’s own subdomain. Not knowing what project to start with I made another interactive story - this time as a little bit of fun for Ruth’s birthday. Happy birthday Ruth!
